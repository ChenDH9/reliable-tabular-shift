
# Data license and redistribution boundaries

No third-party raw file or row-level processed dataset is redistributed. The bundle contains only
official links, checksums, a deterministic rebuild path for one representative ACS shard,
aggregate archived comparison tables, and
article source data. Users must obtain each source under its current official terms, preserve
applicable attribution and no-reidentification obligations, and verify its checksum. The automated
Level B path covers only the official 2018 ACS person PUMS archive. Other tasks require manually
prepared audited Parquet inputs for optional Level C; the bundle does not reconstruct those inputs
from official downloads and does not imply that those sources may be mirrored.
